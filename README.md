# Full Stack Product Catalog Management System

A full-stack web application developed using Spring Boot and React.js for managing products and categories. The application allows users to browse products, filter them by category, search products, and sort product listings through a responsive user interface.

---

# Features

- Product Listing
- Category-Based Filtering
- Product Search Functionality
- Product Sorting
- REST API Integration
- Responsive UI Design
- Full Stack Architecture
